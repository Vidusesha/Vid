    • The total width of an element: Total element width = width + left padding + right padding + left border + right border + left margin + right margin

    The total height of an element: Total element height = height + top padding + bottom padding + top border + bottom border + top margin + bottom margin

    • div { border: 1px solid purple; box-sizing: content-box; height: 250px; margin: 5px 3px 2px 7px; padding: 4px 3px 2px; }

    Total element width = 400 + 3 + 3 + 1 + 1 + 7 + 3 = 418

    Total element height = 250 + 4 + 2 + 1 + 1 + 5 + 2 = 265

    Element width of content area = width - left padding - right padding - left border - right border

    Element width of content area = 400 - 3 - 3 - 1 - 1 = 392

    Element height of content area = height - top padding - bottom padding - top border - bottom border

    Element height of content area = 250 - 4 - 2 - 1 - 1 = 242

Lab 2: 2 HTML Knowledge

HTML tag

html tags are predefined tag which will make up the whole html document. examples:title tag:The document's title is mentioned in the title tag paragragh tag:The content of a paragragh is explained inside the paragraph tag anchor tag:anchor tag is used to define the hypertext link of a html document.

Empty tag

Empty tag is a tag without any content in it and has no closing tag. The major difference between an empty tag and a regular tag is that a regular html tag has a closing tag as well but an empty one will not have one. img and break tags are some of the examples of empty tags

An attribute of an HTML tag

The attribute tag defines the characteristics of an element and contains name and value part in it.

Nesting of tags

Nesting of tags in a legitimate way is very important in a html document.The recent tag which is opened has to be closed first. They are mostly referred to as child elements and sometimes parent when nested tags are added into another nested tag.

id

A unique value assigned inside a HTML document is a id attribute.

id vs class

id is used to describe one single component whereas a class is used to describe more than one element.

type selector

Type selector known as element selectors are also used select a type of element on the page.

head tag

The head part of the html document is used to decribe the head part in an informal way.

title tag,meta tag,link tag,style tag are some of the tags which can go inside the head tag.

body tag

The body tag decribes the body of the HTML document.

Images,lists,tables,text can go inside this tag.
